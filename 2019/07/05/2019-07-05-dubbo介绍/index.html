认识dubbo与dubbo的用途

Dubbo

Apache Dubbo是一个基于Java的高性能开源RPC框架。

特征

  • 基于透明接口的RPC
  • 智能负载均衡
  • 自动服务注册和发现
  • 可扩展性高
  • 运行时流量路由
  • 可视化的服务治理
